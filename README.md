## Description

Ce projet a pour but de créer un programme permettant de générer des fractales en temps réel. Il est possible de se déplacer dans l'espace et de zoomer sur la fractale.

Les fractales disponibles sont les suivantes :

- Mandelbrot
- Julia
- Burning Ship
- Tricorn
- Multibrot

## Commandes

- `Z`: Zoomer sur un point précis de la fractale (Le point est défini dans la classe `Fractal`).
- `R`: Réinitialiser le rendu de la fractale.
